# Fluffy-FiveM

**GitHub release tag:** `APP_VERSION` in the repo root (e.g. `1.6.0` → **v1.6.0**). In-app version: `src/Globals.hpp` — keep both in sync when you ship.

Download the latest **Fluffy-FiveM.exe** and required DLLs from [Releases](https://github.com/9crf69djm5-creator/Fluffy-FiveM/releases), or build locally (`Release | x64`).

**Required files:** `Fluffy-FiveM.exe`, `D3DCompiler_43.dll`, `d3dx9_43.dll`, `d3dx11_43.dll` (same folder as the exe).
